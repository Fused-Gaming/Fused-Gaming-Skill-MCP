# Axe Core

Accessibility testing engine integration for @h4shed ecosystem

## Installation

```bash
npm install @h4shed/tool-axe-core
```

## Development

To develop this tool within the monorepo workspace:

```bash
npm run build --workspace=@h4shed/tool-axe-core
npm run test --workspace=@h4shed/tool-axe-core --if-present
```
