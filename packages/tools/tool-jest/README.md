# Jest

JavaScript testing framework integration for @h4shed ecosystem

## Installation

```bash
npm install @h4shed/tool-jest
```

## Development

To develop this tool within the monorepo workspace:

```bash
npm run build --workspace=@h4shed/tool-jest
npm run test --workspace=@h4shed/tool-jest --if-present
```
