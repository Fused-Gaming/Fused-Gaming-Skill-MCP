# Pa11y

Web accessibility testing CLI integration for @h4shed ecosystem

## Installation

```bash
npm install @h4shed/tool-pa11y
```

## Development

To develop this tool within the monorepo workspace:

```bash
npm run build --workspace=@h4shed/tool-pa11y
npm run test --workspace=@h4shed/tool-pa11y --if-present
```
