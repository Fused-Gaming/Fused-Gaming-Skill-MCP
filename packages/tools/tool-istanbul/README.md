# Istanbul

Code coverage tool wrapper for @h4shed ecosystem

## Installation

```bash
npm install @h4shed/tool-istanbul
```

## Development

To develop this tool within the monorepo workspace:

```bash
npm run build --workspace=@h4shed/tool-istanbul
npm run test --workspace=@h4shed/tool-istanbul --if-present
```
