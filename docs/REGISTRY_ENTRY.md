## Integration Points

### With Claude Desktop

Users can add to Claude Desktop via `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "fused-gaming-mcp": {
      "command": "npm",
      "args": ["run", "dev"],
      "cwd": "/path/to/repo"
    }
  }
}
```
